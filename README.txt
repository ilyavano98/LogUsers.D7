1. Для того чтобы модуль попал в маркетплейс/установленные расширения, требуется скопировать всю директорию loguser.d7 в директорию bitrix/modules
2. Переходим в маркетплейс/установленные решения. Видим модуль с названием "Логирование посещения сайта пользователями", жмем установить.
3. После успешной установки появится соответственное сообщение.
4. Для того чтобы настроить включить/выключить этот модуль с помощью спец настроки, переходим в настройки/настройки продукта/настройки модулей и находим в выпавшем списке "Логирование посещения сайта пользователями"
5. При нажатии попадаем в меню настройки модуля с двумя вкладками. Во первой вкладке "Настройки" имеется возможность включить запись данных или отключить с помощью чекбокса "Включить логирование посещения сайта". Вторая вкладка необязательна в рамках текущей задачи
6. Перемещения в админке записываться в таблицу не будут, а для того чтобы проверить работоспособность достаточно перейти на какой-либо сайт, администрируемый системой битрикс
7. Результат работы запишется в таблицу logusers_d7, которую можно будет найти по пути настройки/производительность/таблицы. В фильре по поиску таблицы по имени ввести "logusers_d7". В ней будут отображаться все требуемые в задаче поля
